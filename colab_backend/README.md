# Satellite Vision - Colab Backend Setup

This backend is designed to run in a single Google Colab cell.

### Running the Backend Script

1.  Open the public Google Colab notebook provided in the web application.
2.  **Run the single script cell.** It will install all dependencies, start the server, and create a secure Cloudflare tunnel.
3.  The script will output a public URL (e.g., `https://...trycloudflare.com`).
4.  Paste this URL into your web app's "Server Config" page to connect the backend.
